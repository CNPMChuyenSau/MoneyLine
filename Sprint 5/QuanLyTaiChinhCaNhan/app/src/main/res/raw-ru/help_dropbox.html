MoneyLine для Android

Синхронизация базы данных программы с десктоп версией MoneyLine для Windows/Linux/Mac с использованием сервиса Dropbox.

Требование: наличие аккаунта dropbox. Если у вас до сих пор нет такого аккаунта - то вы множите создать его пройдя регистрацию на сайте www.dropbox.com.

  • Подключите MoneyLine для Android к сервису Dropbox. Выберите "Соединиться с Dropbox." Если на вашем устройстве установлено официальное приложение сервиса Dropbox, оно автоматически откроется и попросит вас авторизовать доступ к сервису Dropbox приложению MoneyLine для Android. Если у вас на устройстве не установлено офицальное приложение Dropbox, система Android откроет веб браузер на странице входа в сервис Dropbox. После входа на сайт сервиса Dropbox под вашей учетной записью, необходимо будет авторизовать доступ программы MoneyLine для Android к вашему сервису Dropbox.
  • После авторизации MoneyLine для Android, при первом использовании сервиса, следующая папка будет создана автоматически для вас: /Приложении/MoneyLine Mobile для вашего аккаунта Dropbox. Этот каталог нельза создавать вручную !

    Я уже использую базу ПК версии MoneyLine
  • Если у вас уже есть файл с базой данных (mmb) от программы MoneyLine версии для ПК, необходимо скопироватьили переместить этот файл(ы) в каталог /Приложении/MoneyLine Mobile сервиса Dropbox (рекомендуется использовать ПК для этой цели).
  • Из раздела настроек сервиса Dropbox в программе MoneyLine для Android выберите опцию "Файл Dropbox". Приложение откроет новое окно со списком всех файлов в папке /Приложении/MoneyLine Mobile. Выберите требуемый файл для синхронизации.
  • После этого, когда вы вернетесь в настройки Dropbox программы MoneyLine для Android, после примерно 10 секунд программаначнет загрузку выбранного файла, будет показано стандартное уведомление системы Android об окончании загрузки. После того как загрузка завершена, ваше устройство проинформирует вас что файл загружен и готов к использованию в разделе стандартных уведомлений Android, обычно в верхней части экрана и вы должны выбрать опцию "Открыть базу данных" для того, чтобы завершить установления связи между сервисом Dropbox и программой MoneyLine для Android.

    • Я никогда не использовал ПК версию MoneyLine и я хочу синхронизировать базу на мобильном устройстве
  • Если у Вас нет файла mmb MoneyLine десктопной версии, Вы должны нажать "Загрузить БД в Dropbox". После приблизительно 10 секунд, начнется загрузка текущей БД в Dropbox. По окончании данного процесса всплывет уведомление о завершении загрузки, и в папке Dropbox /Приложения/MoneyLine Mobile Вы найдете файл data.mmb.

    • Прочие настройки
  • После того как вы установили связь между программой и сервисом Dropbox, MoneyLine для Android автоматически синхронизирует данные с сервисом Dropbox каждые x минут в соответствии с настройками в разделе "Интервал синхронизации". Если вы не хотите использовать автоматическую синхронизацию, установите значение в ноль для опции "Интервал синхронизации".
  • Важно установить "Загрузка сразу после изменений". Эта установка позволит MoneyLine для Android загрузить БД в Dropbox сразу же после сделанных изменений.
  • Используйте опции "Загрузка из Dropbox" или "Загрузка в Dropbox" для принудительной загрузки базы из или в Dropbox.
  • После установления связи с сервисом Dropbox активируется опция синхронизации в основном меню.

    • FAQ
  • В. Я не могу подключиться к сервису Dropbox или синхронизация не работает ?
    О. В первую очередь вы должны отправить сообщение на этот адрес электронной почты android.money.manager.ex@gmail.com с описанием возникшей проблемы. До решения вашей проблемы, если вы хотите синхронизировать данные программы MoneyLine для Android, вы можете использовать другие приложения для этой цели, такие как FolderSync, Dropsync или найти другое приложение на Google Play.
  • В. Я хочу синхронизировать данные приложения MoneyLine для Android с другими облачными сервисами (Drive, Box.net, Wuala, итд. ..). Как мне это сделать ?
    О. В программе MoneyLine для Android используется встроенный метод синхронизации только с сервисом Dropbox. Если вы хотите синхронизировать файл базы данных с другим онлайн сервисом, необходимо найти приложение на Google Play которое позволит проводить такого рода синхронизацию. База данных программы MoneyManager для Android будет в таком случае сохранена на внешнюю карту памяти устройства и вам будет необходимо открыть ее после синхронизации в программе MoneyLine для Android
  • В. После синхронизации я не вижу своих данных. Означает ли это что данные в базе потеряны ?
    О. Не волнуйтесь, Вы не потеряли данные! В редких случаях возможно, что когда Вы синхронизируете БД, файл загружается некорректно и более не может быть прочитан в MoneyLine для Android. В этих случаях, Вам нужно принудительно загрузить БД из Dropbox. Если БД все еще не открывается, мы рекомендуем Вам удалить файлы из /com.money.manager.ex/dropbox/ и попытаться снова принудительно загрузить файлы из Dropbox.
  • В. Возможно ли восстановить предыдущую версию базы ?
    О. Конечно это возможно. Dropbox сохраняет все версии ваших файлов. Здесь вы найдете руководство сервиса Dropbox о том как восстановить файлы
